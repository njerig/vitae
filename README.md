## Getting Started

Firstly, download packages: `npm install`

Second, set up `next-env.d.ts` and `.env` with correct credentials and keys obtained from repository developers

Third, set up and run the docker container for local development by navigating to `./backend/infra/README.md`

Finally, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can check out [the Next.js GitHub repository](https://github.com/vercel/next.js) - your feedback and contributions are welcome!
